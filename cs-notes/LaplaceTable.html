Laplace Transform Table

A quick-reference table of common Laplace transform pairs and properties used throughout this book. For derivations and worked examples see Chapter 2 — Laplace Transform.

Common Transform Pairs

Signal type Time domain \(f(t)\), \(t\ge 0\) Laplace domain \(F(s)\)
Unit impulse \(\delta(t)\) \(1\)
Unit step \(u_0(t)\) \(\dfrac{1}{s}\)
Unit ramp \(t\) \(\dfrac{1}{s^2}\)
Powers of \(t\) \(t^n\) \(\dfrac{n!}{s^{n+1}}\)
Exponential \(e^{-at}\) \(\dfrac{1}{s+a}\)
\(t\times\) exponential \(t\,e^{-at}\) \(\dfrac{1}{(s+a)^2}\)
\(t^n\times\) exponential \(t^n e^{-at}\) \(\dfrac{n!}{(s+a)^{n+1}}\)
Sine \(\sin(\omega t)\) \(\dfrac{\omega}{s^2+\omega^2}\)
Cosine \(\cos(\omega t)\) \(\dfrac{s}{s^2+\omega^2}\)
Damped sine \(e^{-at}\sin(\omega t)\) \(\dfrac{\omega}{(s+a)^2+\omega^2}\)
Damped cosine \(e^{-at}\cos(\omega t)\) \(\dfrac{s+a}{(s+a)^2+\omega^2}\)
\(t\times\) damped sine \(t\,e^{-at}\sin(\omega t)\) \(\dfrac{2\omega(s+a)}{[(s+a)^2+\omega^2]^2}\)
\(t\times\) damped cosine \(t\,e^{-at}\cos(\omega t)\) \(\dfrac{(s+a)^2-\omega^2}{[(s+a)^2+\omega^2]^2}\)

Properties

Property Result
Linearity \(\mathcal{L}\{\alpha f + \beta g\} = \alpha F(s) + \beta G(s)\)
First derivative \(\mathcal{L}\{\dot{f}\} = sF(s) - f(0)\)
Second derivative \(\mathcal{L}\{\ddot{f}\} = s^2F(s) - sf(0) - \dot{f}(0)\)
\(n\)-th derivative \(\mathcal{L}\{f^{(n)}\} = s^nF(s) - s^{n-1}f(0) - \cdots - f^{(n-1)}(0)\)
Integration \(\mathcal{L}\!\left\{\displaystyle\int_0^t f(\tau)\,d\tau\right\} = \dfrac{F(s)}{s}\)
\(s\)-domain differentiation \(\mathcal{L}\{t^n f(t)\}=(-1)^n\dfrac{d^nF(s)}{ds^n}\)
Frequency shift \(\mathcal{L}\{e^{at}f(t)\} = F(s-a)\)
Time shift \(\mathcal{L}\{f(t-a)\,u_0(t-a)\} = e^{-as}F(s)\)
Time scaling \(\mathcal{L}\{f(at)\} = \dfrac{1}{a}F\!\left(\dfrac{s}{a}\right), \quad a>0\)
Convolution \(\mathcal{L}\{f*g\} = F(s)\,G(s)\)
Periodic signal \(f(t+T)=f(t) \;\Rightarrow\; F(s)=\dfrac{F_1(s)}{1-e^{-sT}}\)
Initial value theorem \(\displaystyle\lim_{t\to 0^+} f(t) = \lim_{s\to\infty} sF(s)\)
Final value theorem \(\displaystyle\lim_{t\to\infty} f(t) = \lim_{s\to 0} sF(s)\)

Caution The Final Value Theorem applies only when all poles of \(sF(s)\) lie in the open left-half plane. If \(F(s)\) has poles on the imaginary axis (other than a simple pole at the origin) or in the right-half plane, the theorem does not give the correct steady-state value.

Concept Formula / Definition

Impulse response \(g(t) = \mathcal{L}^{-1}\{G(s)\}\)
Convolution integral \(y(t) = \displaystyle\int_0^t g(\tau)\,u(t-\tau)\,d\tau\)
